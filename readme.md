## Building

As of right now, Project Special K and [its engine](https://github.com/Kawa-oneechan/beckett) are developed with [Microsoft Visual Studio 2015](https://go.microsoft.com/fwlink/?LinkId=615448). All dependencies should be included.

We welcome any contributions that let this compile on later versions of Visual Studio and other compilers such as clang, and run on operating systems other than Windows, so long as it remains possible to build on VS 2015. The preprocessor is your friend.
